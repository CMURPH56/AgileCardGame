- To build with Maven
 + From current directory run this command line: 
mvn clean package

- To execute the jar file (after successfully build with Maven)
+ From current directory run this command line:
java -jar target\agile-cardgame-1.0-SNAPSHOT-jar-with-dependencies.jar

for windows

java -jar target/agile-cardgame-1.0-SNAPSHOT-jar-with-dependencies.jar
